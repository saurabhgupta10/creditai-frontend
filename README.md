# Credit AI Platform 

## 🔧 Tech Stack
- Frontend: React.js
- Backend: Flask (Python)
- Database: SQLite / MySQL

---

##  Run Project

### Backend
cd backend
pip install -r requirements.txt
python run.py

Server runs on:
http://127.0.0.1:5000

---

### Frontend
cd creditai-frontend
npm install
npm start

App runs on:
http://localhost:3000

---
